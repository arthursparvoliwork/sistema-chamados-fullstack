 🚀 Sistema de Chamados

Sistema web para gerenciamento de chamados técnicos, com dashboard interativo e controle de usuários.

---

 📌 Sobre o Projeto

O Sistema de Chamados foi desenvolvido para permitir:

- 📊 Visualização de métricas em dashboard
- 📝 Abertura e acompanhamento de chamados
- 👤 Controle de usuários
- 🔐 Autenticação com login
- 📈 Dados organizados em banco de dados

Projeto com foco em organização empresarial e experiência profissional.

---

🛠️ Tecnologias Utilizadas

   Frontend
- React
- JavaScript
- CSS

   Backend
- Python
- Flask
- SQLAlchemy

 Banco de Dados
- SQLite (ou MySQL/PostgreSQL se estiver usando)

---

 📊 Funcionalidades

- ✅ Cadastro de usuários
- ✅ Login com autenticação
- ✅ Criação de chamados
- ✅ Alteração de status
- ✅ Dashboard com métricas
- ✅ Integração com banco de dados
